## 🎯 À propos

<div align="center">

> **Innov'Campus** est une plateforme web de valorisation des startups étudiantes, portée par **ESGIS Bénin**.

</div>

Elle offre un espace dédié à l'innovation et à l'entrepreneuriat au sein du campus, permettant aux étudiants de :

- ✨ **Présenter** leurs projets entrepreneuriaux
- 📈 **Gagner en visibilité** auprès de la communauté
- 🤝 **Trouver des partenaires** et des opportunités
- 🏆 **Intégrer l'incubateur** ESGIS

<div align="center">

| 🏆 Startups lancées | 👨‍🎓 Étudiants accompagnés | 🌍 Pays partenaires |
| :-----------------: | :----------------------: | :-----------------: |
|       **15+**       |         **200+**         |        **3**        |

</div>

## 🚀 Installation

### Prérequis

Avant de commencer, assure-toi d'avoir installé :

- ✅ **Serveur web** : Apache ou Nginx
- ✅ **PHP** : version 8.0 ou supérieure
- ✅ **MySQL** : version 5.7+ ou MariaDB 10.3+
- ✅ **Extension PHP** : `mysqli` activée

### Étapes d'installation

<details>
<summary><strong>⚡ Étape 1 : Cloner le dépôt</strong></summary>
<br/>

```bash
git clone https://github.com/Nin-Shinobi/Innov-Campus.git
cd Innov-Campus
```

</details>

<details>
<summary><strong>🗄️ Étape 2 : Créer la base de données</strong></summary>
<br/>

Ouvre **phpMyAdmin** et importe le fichier SQL fourni, ou exécute :

```sql
CREATE DATABASE innov_campus CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

</details>

<details>
<summary><strong>⚙️ Étape 3 : Configurer l'accès MySQL</strong></summary>
<br/>

Modifie le fichier `login_db.inc.php` :

```php
$host     = 'localhost';
$dbname   = 'innov_campus';
$user     = 'root';
$password = '';
```

</details>

<details>
<summary><strong>📁 Étape 4 : Permissions du dossier uploads</strong></summary>
<br/>

```bash
chmod 755 uploads/
```

</details>

<details>
<summary><strong>▶️ Étape 5 : Lancer le serveur</strong></summary>
<br/>

```bash
php -S localhost:8000
```

Ou place le projet dans `C:\xampp\htdocs\` et démarre Apache via XAMPP.

</details>

<details>
<summary><strong>🌐 Étape 6 : Accéder à l'application</strong></summary>
<br/>

Ouvre ton navigateur et va sur :

```
http://localhost:8000
# ou
http://localhost/Innov-Campus
```

</details>

## 🧭 Routes disponibles

<div align="center">

| Paramètre `?pg=` | Page affichée                                             |
| :--------------: | --------------------------------------------------------- |
|    _(aucun)_     | 🏠 **Accueil** — Page d'accueil avec statistiques         |
|      `pro`       | 📋 **Projets** — Liste de toutes les startups avec filtre |
|      `reg`       | ✏️ **Inscription / Modification** — Formulaire projet     |
|      `ins`       | 💾 **Insertion** — Traitement POST du formulaire          |
|      `del`       | 🗑️ **Suppression** — Suppression d'un projet              |
|      `down`      | 📥 **Téléchargement** — Export des données d'un projet    |
|       `ab`       | 👥 **Qui sommes-nous** — Présentation de l'équipe         |

</div>

> 💡 **Exemple :** `index.php?pg=pro` affiche la liste des projets.

## 🔒 Sécurité

> ⚠️ **Ce projet est conçu à des fins éducatives et de démonstration.**
> Avant une mise en production, les améliorations suivantes sont **vivement recommandées** :

<div align="center">

| Problème                    |     Risque      | Correction recommandée                                               |
| --------------------------- | :-------------: | -------------------------------------------------------------------- |
| Injections SQL              | 🚨 **Critique** | Remplacer `sprintf()` par des requêtes préparées (PDO ou MySQLi)     |
| XSS (Cross-Site Scripting)  |  ⚠️ **Élevé**   | Systématiser `htmlspecialchars()` sur toutes les sorties utilisateur |
| Upload de fichiers          |  ⚠️ **Élevé**   | Ajouter une validation MIME côté serveur (`finfo_file()`)            |
| Identifiants MySQL en clair |  ⚠️ **Moyen**   | Utiliser un fichier `.env` ou une config externe hors dépôt          |
| Authentification            |  ⚠️ **Moyen**   | Implémenter `password_hash()` / `password_verify()`                  |
| Headers de sécurité         |  ℹ️ **Faible**  | Ajouter CSP, `X-Frame-Options`, `X-Content-Type-Options`             |

</div>
